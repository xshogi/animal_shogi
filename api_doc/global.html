boardEvaluation(configurations, moves) → {object}

Computes the evaluation of the board based on the heuristic: f(board) = materialScore + mobilityScore Where materialScore = lionWeight * (enemyLion - playerLion) + elephantWeight * (enemyElephant - playerElephant) + giraffeWeight * (enemyGiraffe - playerGiraffe) + chickWeight * (enemyChick - playerChick) + henWeight * (enemyHen - playerHen) where the weights can be set appropriately. mobilityScore = mobilityWeight * (# of enemy moves - # of player moves) where it is generally advantageous to have more moves available.

Name Type Description
configurations list Array of board configurations to test.
